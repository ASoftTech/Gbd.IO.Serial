override int Gbd.IO.Serial.Win32.Settings.SerialBufferSettings.WriteTimeout
getset

Gets or sets the number of milliseconds before a time-out occurs when a write operation does not finish.

WriteTimeout must be either SerialPort.InfiniteTimeout or POSITIVE. a timeout of zero implies that every Write call throws an exception.

The write timeout.
